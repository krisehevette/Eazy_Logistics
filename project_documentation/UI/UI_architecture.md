## 1. Цель документа

---

## 2. Тип рендера

Весь процесс рендеринга расписан в файле "[rendering_pipeline](/project_documentation/UI/rendering_pipeline.md)"

Тип рендера:
**Retained Mode + Scene Graph + Command Buffer Rendering**

- UI представлен деревом объектов, каждый элемент хранит состояние и стиль.
- Последовательный Frame Lifecycle
- Акцент на Batching
- Никаких текстур, кроме шрифтов
