Определить общие параметры пользовательского интерфейса и служить входной точкой для изучения UI.

## 2. Архитектура интерфейса

Архитектура интерфейса, в том числе процесс рендеринга подробно описаны в документе "[UI_architecture](/project_documentation/UI/UI_architecture.md)"

---

## 3. Настройки интерфейса

- **Соотношение сторон:** 16:9  
- **Разрешения:** 1920×1080 и 1366×768  
- **FPS:** фиксированные 60 кадров  
- UI полностью 2D, ось Z используется как порядок слоёв
