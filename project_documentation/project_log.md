# История проекта.


## День 1

- Определена архитектура: **модульная + событийная + контракты + SOLID**.
- Создан черновик **Architecture Overview**.
- Настроен Python‑скрипт для автогенерации структуры проекта.

---

## День 2.

- Определены фундаментальные принципы UI и тип рендера
- Созданы черновики **UI_context, UI_architecture, rendering_pipeline**
- Настроен Python‑скрипт для упаковки документации проекта в единый all_doc.md

---

## День 3.

- Сформирована фундаментальная структура ядра программы
- Созданы черновики **core_context** и **event_bus**
- Создано slnx решение проекта и **main.cpp**, организованна структура проекта в соответствии с этими изменениями

---

## День 4.

- Определена архитектура **инициализации приложения**
- Создана **общая схема инициализации** в `application_initialization.md`, сформированы 7 логических этапов инициализации
- Создан и оформлен документ **application_startup**, описана точка входа `main.cpp`, определена роль `Application::Run()` как центрального оркестратора
- Сформирована финальная структура папок для кода инициализации: `src/Core/Application`, `src/Core/Initialization`, `src/Core/MainLoop`, `src/Core/Shutdown`.

---

## День 5.

- Сформирована фундаментальные элементы ядра программы
- Созданы документы для **EventBus**, **Config**, **Logger**, **ModuleRegistry** и **ApplicationState**
- Сформирован второй этап инициализации программы: Инициализация Ядра в документе `core_init.md`
